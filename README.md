# Тестовое задание Ecom.tech

Одностраничное приложение для отображения и поиска товаров.

## Запуск проекта

```bash
npm install
npm run dev
```

Ссылка [http://localhost:3000](http://localhost:3000)

API [http://localhost:3000/api/products](http://localhost:3000/api/products)

## Технологии

- Next.js 
- React
- TypeScript
- Tailwind CSS
